# 🔷 ChainProbe

### Real-Time On-Chain Intelligence Engine

ChainProbe is a modular blockchain transaction analysis toolkit that evaluates wallet behavior and generates a dynamic risk score (0–100).

It is designed as a foundation for building:

- Crypto fraud detection tools
- Wallet monitoring systems
- Compliance & AML research tools
- On-chain analytics dashboards
- Web3 security applications

---

## 🚀 Key Features

- Behavioral wallet analysis
- Transaction frequency monitoring
- Large transfer anomaly detection
- Suspicious interaction tracking
- Statistical deviation scoring
- Modular risk engine
- API-ready architecture

---

## 🧠 How It Works

ChainProbe processes wallet transaction data and extracts behavioral metrics such as:

- Total transaction count
- Average transaction value
- Transaction amount deviation
- Maximum transfer size
- Interactions with flagged addresses

These metrics are evaluated through a configurable heuristic scoring engine to produce a normalized risk score.
